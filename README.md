# つぶやきメモ

ブラウザのローカルストレージでデータの管理・保存が可能なSNSライクなメモ帳アプリケーションです。

本アプリケーションは以下のURLからご利用いただけます。

- [https://github.com/mizusameblue/tsubuyaki-memo.git](https://github.com/mizusameblue/tsubuyaki-memo.git)

## 機能概要

### 1. ローカルストレージによるデータ保存

投稿データはすべてブラウザのローカルストレージ（localStorage）に保存されます。外部サーバーを使用しないため、データが外部に送信されることはありません。

### 2. タイムラインとスレッド表示

- **返信機能**：投稿に対して返信を紐付けることができ、会話履歴をツリー構造で確認できます。
- **編集・削除機能**：投稿した内容をその場で修正したり、個別に削除したりすることができます。
- **ソート機能**：タイムラインの表示順序を「新しい順」と「古い順」にいつでも切り替えることができます。
- **自動リンク変換**：本文内のURLはハイパーリンクに変換され、ハッシュタグも自動的に検出されてクリック可能になります。

### 3. フィルタリングと検索

目的の投稿を探すための表示モードと検索機能を備えています。

- **すべて**：全ての投稿を表示します。
- **いいね**：お気に入り登録した投稿のみを表示します。
- **タグ**：本文内のハッシュタグを抽出し、タグごとの件数付きチップとして一覧化します。
- **月別**：投稿年月ごとの件数をセレクトボックスに集計し、月ごとに絞り込みます。
- **キーワード検索**：入力されたキーワードやハッシュタグによるリアルタイム検索が可能です。

### 4. インポート・エクスポート

- **エクスポート**：全投稿データをJSON形式のファイルとしてダウンロードできます。
- **インポート**：エクスポートしたJSONファイルを読み込みます。既存データと重複しないものだけを自動で統合します。

## 使用方法

### 1. 投稿・返信・編集

- **新規投稿**：画面上部の入力欄にテキストを入力し、「つぶやく」ボタンを押します（最大1,000文字）。入力中のテキストは下書きとして自動保存されます。
- **返信**：投稿下部の「返信」ボタンを押すと入力欄が展開し、スレッドを作成できます。
- **編集・削除**：各投稿の「編集」ボタンから内容の修正、「削除」ボタンから個別の削除が可能です。

### 2. 検索と絞り込み

- 画面中央のタブ（すべて / いいね / タグ / 月別）で表示モードを切り替えます。
- 「タグ」または「月別」タブでは、上部に表示される一覧やセレクトボックスから項目を選択して絞り込みます。
- 検索窓にキーワードを入力すると、現在の表示モード内でリアルタイムに検索が行われます。

### 3. 並び替えと一括削除

- タイムライン右上のセレクトボックスより、「新しい順」と「古い順」を切り替えることができます。
- 各投稿のチェックボックス、または「すべて選択」にチェックを入れることで、選択した複数の投稿を一括削除できます。

### 4. データのバックアップ

- 画面右上の「エクスポート」ボタンからデータをファイルとして保存します。
- データを復元・移行する場合は、「インポート」ボタンから過去に保存したJSONファイルを選択します。

## 開発

このアプリケーションのドキュメントおよびコードの一部の作成には、支援ツールとしてGeminiが使用されています。
